# Manuel Utilisateur

## Patient
- Créer un compte
- Prendre rendez-vous
- Consulter résultats
- Télécharger résultats

## Médecin
- Consulter patients
- Lire résultats d’analyses
- Valider examens

## Laborantin
- Voir examens à réaliser
- Saisir résultats
- Vérification auto normes

## Secrétaire
- Gérer patients
- Gérer rendez-vous
- Gérer paiements

## Admin
- Gérer utilisateurs et rôles
- Gérer analyses & normes
- Consulter statistiques
